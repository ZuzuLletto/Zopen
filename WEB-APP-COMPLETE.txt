🚀 TO GET STARTED
-----------------

1. Open Terminal/Command Prompt

2. Navigate to project:
   cd c:\Users\Zuzu\Desktop\Projeler\caseopensim\web-app

3. Install dependencies:
   npm install

4. Add your images to:
   /public/assets/cases/   (3 case images)
   /public/assets/skins/   (15 skin images)

5. Run development server:
   npm run dev

6. Open browser:
   http://localhost:3000
